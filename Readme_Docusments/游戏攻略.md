## 一、基础机制速览

### 时间系统

一天 = **8 回合**（凌晨→早上→上午→中午→下午→傍晚→夜晚→深夜）

### 核心属性

| 属性          | 说明                         | 归零惩罚      |
| ----------- | -------------------------- | --------- |
| ❤️ 生命值 (HP) | 初始 240，上限 240              | 死亡        |
| 🍖 饱食度      | 初始 100，每回合 -4              | 每回合 -10HP |
| 💧 水分       | 初始 100，每回合 -4              | 每回合 -10HP |
| 🦠 感染值      | 被丧尸攻击+5，上限 100             | 达到 100 死亡 |
| 💔 崩溃值      | 超过 50 显示"崩溃"，每 16 回合自动 +20 | 影响状态显示    |

### 新手必知

- **无怪图"曙光阵地"可无限生存**——睡觉、吃饭、喝水、打工全在这里
- **每天可打工 4 次**（白天才能打，晚上不能），每次 +5 崩溃、-10 水分、-10 饱食
- **每天可乞讨 1 次**（跟阵地 NPC 对话有几率要给吃的）
- **谷仓每 3 天可采摘 3 个水果**
- **瞭望塔登塔可减崩溃**
- **江边港口看景色也可减崩溃**
- **睡眠跳过 4 回合，恢复约 20\~40 HP + 减少崩溃**

## 二、快速上手：无限苟活打工仔流

> 适合：不想冒险、想安稳发育的玩家
> 难度：★☆☆☆☆（基本不会死）

### 核心循环（一天8回合）

```
打工(1回合) → 打工(2回合) → 打工(3回合) → 打工(4回合)
→ 乞讨/对话(5回合) → 吃饭喝水(6回合) → 睡觉(7-8回合)
```

### 消耗与收益计算

| 项目           | 消耗   | 补充方式            |
| ------------ | ---- | --------------- |
| 崩溃 +20（打工4次） | 睡觉恢复 | 睡觉一次 -20\~30 崩溃 |
| 水分 -40（打工4次） | 喝水   | 乞讨的水果/阵地搜刮      |
| 饱食 -40（打工4次） | 吃饭   | 谷仓3天9个水果、阵地食物   |

**结论：** 打工 4 次赚的物资远高于消耗，只要不出去打架，**永远不会死**。

### 打工路线图

1. 每天起床先 **乞讨**（阵地 NPC 对话有概率给水果）
2. **打工 4 次**（注意：黑夜不能打工）
3. 用赚来的物资 **吃饭喝水**
4. **睡觉** 跳过夜晚，恢复崩溃和血量
5. 每 3 天去一次 **谷仓采摘水果**
6. 攒够一定实力再去探索

## 三、完整攻略·新兵到老兵

### 第一阶段：开局求生（1\~5天）

**目标：** 活下来 + 拿到基础武器

#### 第1天

1. ✅ **去曙光阵地找 V 小姐对话**，领取初见任务（1食物+1饮料）
2. ✅ 完成任务获得 **小匕首（40伤害）**——你的第一把像样的武器
3. ✅ 去**乡村废弃谷仓**（★轻松，遇敌率仅3%）搜刮食物和武器
4. ✅ 注意血量，不舒服就回家睡觉

#### 第2\~5天

- 🔄 每天循环：打工 → 吃饭 → 睡觉
- 🏪 每3天去谷仓**采摘水果**（稳定食物来源）
- 🎒 去**河边露营地**（★★低危）搜刮背包，目标是**28格登山包**以上
- ⚔️ 谷仓有15%概率掉落近战武器，搜到\*\*铁铲(48伤)/撬棍(60伤)\*\*就换掉小匕首

#### 武器过渡路线

```
拳头(10) → 小匕首(40) → 铁铲(48)/撬棍(60)/消防斧(68) → 指虎刀(66)/大砍刀(75)
→ 西洋剑(79)/龙抄剑(88)/长骑枪(90) → 武士刀(99)/唐横刀(102)/电锯(105) → 寰宇之剑(118)
```

### 第二阶段：积累期（5\~15天）

**目标：** 攒物资、升装备、准备救伙伴

#### 推荐刷图顺序

| 地图      |  难度 | 遇敌率 | 主要产出     |
| ------- | :-: | :-: | -------- |
| 乡村废弃谷仓  |  ★  |  3% | 食物+武器    |
| 深山农家乐村落 |  ★  | 10% | 食物+水果+种子 |
| 河边露营地   |  ★★ | 12% | 背包+药品    |
| 高校大学城   |  ★★ | 15% | 食物+背包    |
| 城郊废弃加油站 |  ★★ | 18% | 弹药+武器    |

#### 关键行动

1. 🎒 **找大背包**：露营地搜刮尸体（5%背包率）、瞭望塔（10%背包率，**出56格超大旅行背包**）
2. ⚔️ **刷武器**：加油站（15%近战）、工业园区（30%近战）——**工业园区是刷近战武器的圣地！**
3. 💊 **囤药品**：重点囤**抗感染血清**和**止血带**（后面任务大量需要）
4. 🚬 **攒香烟**：香烟是硬通货！莉莉丝回收武器给烟，黑市商人用烟换一切

#### 打不过怎么办？

- 血量不健康就**回家睡觉**，别硬撑
- 崩溃高了去**瞭望塔登塔**（减崩溃）或**江边港口看景色**
- 带足食物和水再出门

### 第三阶段：伙伴招募（15\~30天）

**目标：** 30天内救下柳如烟 + 招募露露薇

> ⚠️ **柳如烟有时限：30天后她会彻底变成丧尸，无法再救！**

#### 前置：获得60+伤害武器

- 去**谷仓/加油站/工业园区**刷武器
- 或者找 V 小姐做任务拿好武器

#### 救柳如烟（限时！）

```
市中心综合商场（★★★中危）
  → 找到半感染女人（柳如烟）
  → 需要 3支抗感染血清 才能救她
  → 成功救下 → 触发长剧情 → 带回家 → 获得第一个伙伴
```

**血清获取途径：**

| 方式                | 说明              |
| ----------------- | --------------- |
| 🏥 城郊大型医院（★★★★高危） | 40%概率掉药品，刷得越快越好 |
| 🛒 黑市商人           | 随机刷新，偶尔有血清      |
| 💊 搜刮中高难度图        | 警察局、研究所等会出      |

**刷医院技巧：**

- 别贪，打一轮就回家睡觉，睡醒再来
- 带足食物和水
- 有60+伤害武器再去，不然打不过军警丧尸（160HP/36伤）

#### 招募露露薇（无时限）

```
城郊大型医院（★★★★高危）
  → 找到理智的护士丧尸（露露薇）
  → 投喂罐头食品（黄桃/豆类/牛肉/军粮/沙丁鱼/金枪鱼/秋刀鱼/青花鱼罐头）
  → 好感度达到150 → 带回家 → 获得第二个伙伴
```

**技巧：** 罐头在谷仓、超市、农家乐都能刷到。每次投喂+2\~3好感，大概需要50\~75个罐头。

#### 伙伴作用

|      伙伴     | 获取条件       | 每日产出                  |
| :---------: | ---------- | --------------------- |
|    🌸 柳如烟   | 3血清（限30天）  | 每日2随机物资（食物/饮料/水果/弹药）  |
|    🏥 露露薇   | 150好感（不限时） | 每日1药品（创可贴\~肾上腺素）      |
|  ⚔️ 柳如烟（战斗） | 已招募        | 战斗中30%概率协同攻击·造成20额外伤害 |
| 💉 露露薇（战斗后） | 已招募        | 战斗胜利后自动回血+20          |

**香烟就是末日货币！** 所有高级交易都需要香烟。

**刷香烟方法：**

4. 莉莉丝回收远程武器（2烟/把）

### 🏥 刷医院生存指南

医院是游戏最重要的地图（出血清救柳如烟），但也是高危图（★★★★）。

### 🏹 远程武器选择指南

#### 弓弩 vs 枪械

| 对比项 | 弓/弩 | 枪械 |
| :---- | :---- | :--- |
| 暴击率 | 0% | 10%\~45%（视武器而定） |

### 💱 经济循环指南

香烟是末日的硬通货，掌握经济循环是长期生存的关键。

#### 核心循环：香烟 → 物资 → 香烟

```
搜刮武器 → 莉莉丝回收 → 获得香烟
    ↓
香烟 → 黑市商人 → 买弹药/武器/种子
    ↓
更好的武器 → 挑战高危图 → 更多武器/物资
    ↓
多余武器 → 莉莉丝回收 → 更多香烟
    ↑_________________________|
```

#### 香烟获取途径汇总

| 途径 | 收益 | 稳定性 |
| :--- | :--- | :---: |
| 莉莉丝回收近战武器 | 1烟/把 | ⭐⭐⭐ 后期武器溢出，稳定变现 |
| 莉莉丝回收远程武器 | 2烟/把 | ⭐⭐⭐ 远程更值钱 |

| 击杀黑市商人 | 2\~5烟 | ⭐ 有风险 |
| 城堡打工 | 1烟/天 | ⭐ 消耗大(70饱食+70水分+49崩溃) |
| 莉莉丝150好感奖励 | 20烟(一次性) | ⭐⭐⭐ 一次性大额 |

#### 香烟消费优先级

1. **清除感染(10烟)** — 感染值接近100时必须清，否则死亡
2. **维修枪支(10烟)** — 主力武器坏了很危险

**省钱技巧：** 打小怪用弓弩（零维修费），打BOSS/强敌才用枪械，最大化香烟利用率。

## 十、常见问题

**Q：开局怎么玩？**
A：去谷仓（安全）搜刮→找V小姐拿小匕首→打工攒钱→逐步探索低危图。

**Q：柳如烟没救到怎么办？**
A：30天以后她会彻底变成丧尸，只能读档或者重新开局。所以**一定要在30天内凑够3支血清**。

**Q：背包满了怎么办？**
A：造仓库（基地建设→建造仓库），把不用的物资存进去。

**Q：怎么快速刷好感度？**
A：每天对话4次（每次+1\~3好感，30%概率额外+1），送她需要的任务物品。

**Q：感染值高了怎么办？**
A：用清创药(-25)/抗生素(-25)/抗感染血清(-60)，或者找莉莉丝花10烟清除-50感染。

**Q：崩溃值高了怎么办？**
A：睡觉（-20\~30）、吃止痛片(-5)/肾上腺素(-10)/手术包(-10)/战地医疗箱(-30)、去瞭望塔登塔、去江边看景色。

**Q：近战武器坏了怎么办？**
A：找莉莉丝维修？不对，莉莉丝只修**远程武器**。近战武器坏了就换一把新的。好在近战武器掉落很多。

**Q：远程武器完整度低了怎么办？**
A：找莉莉丝花10烟维修(枪械)或5烟维修(弓弩)。

**Q：最强的武器是什么？**
A：近战是寰宇之剑(118伤/18%连击)，远程是AWM(200伤/45%暴击·传奇级)。不过AWM正常游戏无法获得。

**Q：游戏有结局吗？**
A：击败尸王·寂灭后没有特殊结局，这是开放式的生存游戏，可以继续玩下去。

**Q：AWM怎么获得？**
A：AWM(200伤/45%暴击/传奇级)是游戏最强远程武器，只能通过**绝密航天基地**5%概率发现坠毁太空舱获得（仅1次机会）。绝密航天基地遇敌率高达90%，建议带满药品和顶级装备再去。正常游戏极难获得，属于终极收藏品。

**Q：怎么加入曙光阵地？**
A：在曙光阵地找到**阵地首领**，选择"加入阵地"即可获得**曙光徽章**。注意：持有贵族身份牌时无法加入（首领会拒绝），需要先注销贵族身份。

**Q：贵族身份有什么用？**
A：贵族身份牌(50烟)可以进入**末日城堡内部**，享受以下特权：参加贵族宴会(回满饱食+水分)、参加贵族舞会(崩溃-50)、在贵族客房休息(HP+50)。但持有贵族身份会失去曙光阵地的乞讨和打工权限，且无法加入曙光阵地。

**Q：城堡债务逾期会怎样？**
A：逾期未还(超过10天)会触发**没收全部物品**——身上所有东西被扒光！这个惩罚只触发一次，还清债务后恢复正常。建议借贷前确保10天内能还清，或在逾期前找银行行长宽限5天。

**Q：双身份冲突怎么处理？**
A：游戏**不允许同时持有曙光徽章和贵族身份牌**。如果你已有曙光徽章再去办贵族身份，系统会自动清除贵族身份牌、保留曙光徽章。如果已有贵族身份牌去加入曙光阵地，阵地首领会拒绝互动。建议先确定走哪条路线再选择身份。
