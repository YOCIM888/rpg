> 本文档仅供开发者测试使用，游戏内无任何提示

## 二、属性修改指令

### 1. 生命值 (health)

格式：`/get_health_数值`

增加玩家生命值，最大上限240点。

**示例：**
```
/get_health_100   // 生命值+100
/get_health_50    // 生命值+50
/get_health_240   // 生命值+240（回满）
```

### 2. 完全恢复 (full)

格式：`/get_full_1`

立即恢复所有属性到满值：
- 生命值 → 240
- 饱腹度 → 100%
- 水分 → 100%
- 感染值 → 0%
- 崩溃值 → 0%

**示例：**
```
/get_full_1   // 恢复所有属性
```

### 3. 天数推进 (day)

格式：`/get_day_数值`

增加游戏天数。

**示例：**
```
/get_day_10   // 时间快进10天
/get_day_30   // 时间快进30天
```

## 四、注意事项

1. **数量限制**：每次最多获取999个物品
2. **背包容量**：物品会占用背包格子，背包满了无法获取
3. **武器耐久**：近战武器获取时自动满耐久
4. **弹药单位**：弹药命令每单位为1发，但游戏内弹药以叠堆存储
5. **建筑材料**：使用 `building` 类型，存入 `其他` 背包分类
6. **隐蔽性**：作弊命令失败时无任何提示，成功时显示物品名称和数量
7. **特殊物品互斥**：贵族身份牌与曙光徽章不可同时持有，同时获得时贵族身份牌自动销毁

## 八、常见问题

**Q：为什么有些命令没反应？**

A：物品名必须完全匹配，包括中文标点。例如"抗感染血清"不能写成"抗感染药剂"，"9×19mm"中的"×"是乘号不是字母x。命令失败时不会有任何提示。

**Q：远程武器命令怎么输入？**

A：远程武器使用**武器ID**（如AWM、G17、UZI），不是中文名。弓弩类除外，可用中文名（如"木弓"、"狙击弩"）。

**Q：弹药命令用简写还是全名？**

A：两种都支持。简写更方便：`9mm`、`magnum`、`556`、`762`、`76251`、`300`、`shotgun`、`arrow`。全名也可以：`9×19mm`、`7.62×39mm`等。

**Q：背包满了怎么办？**

A：先升级背包（`/get_backpack_超大旅行背包_1`），或者建造仓库把不用的物资存进去。背包满了之后作弊命令添加的物品会丢失。

**Q：贵族身份牌和曙光徽章能同时拥有吗？**

A：不能。两种身份互斥，同时获得时贵族身份牌会自动销毁。请根据需要的路线选择一种。

**Q：寂灭徽章有什么用？**

A：寂灭徽章是击败最终BOSS"尸王·寂灭"后的荣誉纪念品，目前无特殊功能，作为通关证明。
