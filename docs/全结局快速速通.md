可继续结局触发后选择"继续生存"即可继续游戏。
Game Over结局需要存档切换：
1. 在触发Game Over结局前存档
2. 触发结局（结局被记录到unlockedEndings）
3. 读档回到触发前
4. 走另一条路线触发下一个结局

## 周目一：曙光路线（6结局）

**目标**：captain + farming + love_hate + elopement + food + death
**预估天数**：约120-150天
**阵营**：曙光阵地

### 第一阶段：基础建设（Day 1-30）

**Day 1-3：开局生存**
- 选择探索低危地图（河边露营地、乡村废弃谷仓）
- 收集基础物资：食物、水、药品
- 目标：存活并积累初始资源

**Day 4-10：救柳如烟（30天时限！）**
- 前往市中心综合商场 → 半感染的女人
- 需要3支抗感染药剂
- 如果血清不够，去城郊大型医院探索获取
- ⚠️ 必须在30天内完成，否则柳如烟变丧尸
- 💡 发现柳如烟后，生存笔记中会自动添加救援倒计时提醒
- 救出后：liuruyanRescued = true，柳如烟每天可搜集物资

**Day 5-15：刷NPC好感度**
- 每天与V小姐、苏小涵、莉莉丝各对话4次（+4/人/天）
- 送医疗品（+5/次，莉莉丝+10/次）或食物/饮品（+3/次，莉莉丝+6/次）
- 送香烟（+2/次）
- 优先级：莉莉丝 > 苏小涵 > V小姐（莉莉丝加成最高）
- 刷满150约需7天/人，三人并行约7-10天

**Day 10-20：击杀关键BOSS**
- 市中心综合商场 → 连锁大型仓储超市 → 击杀黑影
  - 黑影属性：HP 200，伤害25-45，有远程，闪避20%
  - 击杀后：shadowDefeated
- 丧尸巢穴 → 沉默的丧尸之王
  - 丧尸之王属性：HP 500，伤害50-80，有远程，闪避25%
  - 击杀后：zombieKingDefeated
- 城郊大型医院 → 救出露露薇（投喂罐头至好感150）
  - nurseZombieRescued

**Day 15-25：击杀马三**
- 前往国道高速服务区
- ⚠️ 注意：这里选择击杀马三（而非完成任务链），因为ending_newforce需要outlawKilled
- 马三属性：HP 300，伤害40-60，有远程，闪避40%
- 击杀后：outlawKilled
- 💡 ending_food（马三任务链）将在存档切换中单独完成

### 第二阶段：曙光队长（Day 25-60）

**Day 25-60：刷首领好感至150**
- 每天与首领对话1次（+1）+ 送礼1次（+1）= +2/天
- 完成首领任务链获取大量好感：
  - 任务1（录音笔）：+10
  - 任务2（回信+5白酒）：+20
  - 任务3（3万能针剂）：+30
- 任务奖励合计+60，剩余90需45天对话+送礼
- 💡 优先完成任务链加速好感

**首领任务1：录音笔**
- 前往河边露营地 → 尸臭的帐篷
- ⚠️ 需要曙光队长徽章才能进入！但队长徽章需要首领好感150...
- 🔄 解决方案：先刷好感到150→获取队长徽章→再进帐篷拿录音笔
- 或者：录音笔也可以在其他途径获取（需确认）

**首领任务2：回信+5白酒**
- 拿到录音笔后，首领会给"给妹妹的信"
- 需要前往末日城堡找皇后，获取"皇后的回信"
- 还需要5瓶高度白酒
- ⚠️ 进入城堡需要城堡身份牌（25香烟），但持有曙光徽章时无法获得城堡身份
- 🔄 解决方案：先退出阵地→办城堡身份→见皇后拿回信→退出城堡身份→重新加入阵地
- 或者：不完成首领任务2，纯靠对话+送礼刷好感（更慢但更简单）

**首领任务3：3万能针剂**
- 万能针剂获取方式：
  - 柳如烟任务1（高校大学城→旧教室→医务室）获得1支
  - 其他途径需要探索获取
- 完成后获得：定情信物(love_token) + 9×19mm×100

**获取曙光队长徽章**
- 条件：V/小寒/莉莉丝/首领好感均≥150
- 与首领对话选择"关于队长徽章"
- 获得：dawn_captain_badge
- 🏆 **触发ending_captain** → 选择"继续生存"

### 第三阶段：沐苗苗+皇后（Day 60-100）

**沐苗苗任务链**
- 沐苗苗在曙光阵地内，无需外出
- 好感度获取：照看实验田（+1~3/天）+ 送礼
- 刷到150后触发"秘密"→ 获取农业大师徽章
- 任务1：3胡萝卜 + 3土豆
- 任务2：5高度白酒 + 10黑麦
- 任务3：15种不同作物各1
  - 💡 作物从沐苗苗处购买种子，在实验田种植收获
  - 需要较长时间（每种作物需1-3天成熟）
- 完成任务3后：获得原野之弓+箭矢×30
- 🏆 **触发ending_farming** → 选择"继续生存"

**皇后任务链（需要定情信物）**
- 前提：持有首领任务3给的定情信物
- ⚠️ 需要城堡身份才能进入城堡见皇后
- 操作：退出阵地→办城堡身份（25香烟）→见皇后→完成任务→退出城堡身份
  - 退出阵地后好感度保留，队长徽章会失去，但ending_captain已触发
- 任务1：5百味啤酒 + 5动力啤酒 → 皇家币×10
- 任务2：10枸杞 → 皇家币×20
- 任务3：3西洋剑 → 皇家币×30
- 任务4：1把M700 → 皇家币×40 + 城堡通行证
  - 💡 M700需要从博士交易获取（30血清），或从其他途径获取
- 🏆 **触发ending_love_hate** → 选择"继续生存"

### 第四阶段：存档切换Game Over结局（Day 100-120）

**存档位准备**
- 在完成所有可继续结局后，存档到存档位A

**ending_elopement（私奔）**
1. 前往城郊废弃加油站找王铁柱，5香烟换1汽油，兑换10次（需50香烟）
2. 前往高校大学城 → 一间旧教室
3. 完成柳如烟任务1-3：
   - 任务1：一起去医务室 → 获得万能针剂
   - 任务2：去体育场 → 击败橄榄球丧尸（HP250/30-55伤/15%闪避）
   - 任务3：装备撬棍 → 开门获得车钥匙
4. 任务4：消耗10汽油
5. 🏆 **触发ending_elopement** → Game Over
6. 💡 结局已记录，读档回到存档位A

**ending_food（沦为盘中餐）**
1. 读档存档位A
2. 前往国道高速服务区 → 服务区餐厅 → 食用杂肉（hasEatenStrangeMeat = true，⚠️ 杂肉每次+10感染值）
3. 与马三对话 → "关于餐厅的食物"
4. 完成马三任务1-3：
   - 任务1：3瓶动力啤酒
   - 任务2：3份作物
   - 任务3：10根香烟
5. 任务4：选择"接受"（⚠️ 不可逆选择，无论接受还是击败马三都会触发终局）
6. 🏆 **触发ending_food** → Game Over
7. 💡 结局已记录，读档回到存档位A

**ending_death（死亡）**
1. 读档存档位A
2. 故意让HP归零或感染度到100
3. 🏆 **触发ending_death** → Game Over

**周目一完成！已解锁6个结局。**

## 周目二：城堡+火箭+999路线（6结局）

**目标**：999 + newforce + prince + hope + space + stay
**预估天数**：999+天（最耗时的周目）
**阵营**：先曙光后城堡

### 关键策略：先刷好感再转阵营

ending_hope需要V/小寒/莉莉丝好感150，但持有城堡身份时无法与她们交互。解决方案：**先加入曙光刷满好感→退出阵地→转城堡线**。好感度在退出阵地后保留！

### 第一阶段：刷好感+转阵营（Day 1-20）

**Day 1-10：加入曙光，刷满NPC好感**
- 加入曙光阵地
- 刷V小姐、苏小涵、莉莉丝好感至150（约7-10天）
- 💡 不需要刷首领好感（后续会杀首领）

**Day 10-15：退出阵地**
- 与首领对话选择"退出阵地"
- 曙光徽章和队长徽章被移除
- 好感度保留！

**Day 15-20：办理城堡贵族身份**
- 前往末日城堡 → 办理贵族身份（25香烟）
- 获得城堡身份牌

### 第二阶段：国王任务1-4（Day 20-50）

**国王任务1：10个葡萄糖服液 → 子爵**
- 葡萄糖服液从探索医院或交易获取

**国王任务2：10瓶高度白酒 → 伯爵**
- 高度白酒从探索或交易获取

**国王任务3：10香烟+5百味啤酒+5动力啤酒 → 侯爵**
- 香烟和啤酒从探索或交易获取

**国王任务4：3支万能针剂 → 公爵**
- 万能针剂获取：柳如烟任务1给1支，其余需探索
- 💡 柳如烟任务1-3可以在此时完成（不需要曙光身份）

### 第三阶段：博士交易+火箭前置（Day 30-60）

**博士交易：30支抗感染血清 → M700+7.62×51mm×30**
- 血清从探索医院、莉莉丝交易、皇后杂货铺获取
- 💡 30支血清是大数目，需要持续收集

**击败巨型丧尸傀儡**
- 前往绝密航天基地 → 发射中心
- 傀儡属性：HP 400，伤害35-60，有远程，闪避20%
- 击败后：giantPuppetDefeated

**接受火箭维修咨询**
- 与博士对话选择关于火箭
- 💡 如果前置条件未满足，游戏内会在博士的对话选项中显示具体缺失条件

### 第四阶段：生存至999天（Day 60-999）

**🏆 这是整个速通最耗时的阶段**

**高效生存策略**：
- 每天在曙光阵地工作获取资源（不需要曙光身份也能工作？需确认）
- 或在基地休息+外出探索低危地图收集物资
- 保持饱食/水分/健康，避免死亡
- 利用柳如烟每日搜集物资
- 定期去能源开采井采集纯净能源（博士任务1完成后）

**Day 999：触发ending_999**
- 🏆 **触发ending_999** → 获得**次元收纳背包（68格）** → 选择"继续生存"
- 💡 次元背包是火箭链地下区域的关键道具！

### 第五阶段：火箭链（Day 999+）

**获取AWM（5%概率太空箱）**
- 反复探索绝密航天基地
- 每次探索有5%概率触发太空箱
- 💡 这是火箭链最大的随机瓶颈，可能需要大量探索

**国王任务5：暗杀首领 → 储君**
- 与国王对话选择"清除异己"
- 前往曙光阵地暗杀首领
- ⚠️ 首领战斗：HP 300，伤害40-60
- 完成后获得储君身份牌(crown_prince_id)
- 🏆 **触发ending_prince** → 选择"继续生存"

**获取地下钥匙**
- 条件：储君(rank 6) + 已接受博士任务1 + 上缴AWM
- 与国王对话选择"陛下，我想去地下区域"
- 交出AWM → 获得地下钥匙

**拆除核电装置**
- 进入城堡地下区域
- 需要装备：大扳手（近战武器）+ 次元收纳背包（68格，来自ending_999）
- 拆除后获得小核能发电机
- 交给博士 → 博士任务1完成 → 解锁能源开采井

**博士任务2：15个纯净能源**
- 从能源开采井每天采集1个
- 需要15天
- 💡 采集时游戏会显示当前进度（X/15）
- 💡 如果之前已经开始采集，可能已经攒够了

**博士任务3：10医疗+10食物+10饮品**
- 准备物资交给博士
- 火箭维修完成！

**ending_newforce自动触发**
- 检查条件：nurseZombieRescued + liuruyanRescued + outlawKilled + shadowDefeated + zombieKingDefeated + doctorQuest3Done
- ⚠️ outlawKilled：周目一中杀了马三，但这是新周目！需要在本周目重新击杀马三
- 💡 确保本周目也完成了所有6个条件
- 💡 生存笔记中会追踪新生力量结局进度（X/6），完成任意前置条件后即可查看
- 🏆 **触发ending_newforce** → 选择"继续生存"

### 第六阶段：火箭三选一（存档切换）

**准备存档**
- 在发射中心选择前存档到存档位B

**ending_hope（希望的火种）**
1. 选择"带上重要的人"
2. V/小寒/莉莉丝好感已满150（第一阶段刷的）
3. 💡 游戏会显示当前好感度进度（如 V:150/150 小涵:150/150 莉莉:150/150）
4. 🏆 **触发ending_hope** → Game Over
5. 读档存档位B

**ending_space（飞向太空）**
1. 选择"自己登上火箭"
2. 🏆 **触发ending_space** → Game Over
3. 读档存档位B

**ending_stay（留下来吧）**
1. 选择"我不走，博士走吧"
2. 🏆 **触发ending_stay** → 选择"继续生存"
3. 💡 博士留下改良抗体针制造仪器，每天可领取1支

**周目二完成！已解锁6个结局。**

## 物品速刷指南

### 香烟（全流程最缺资源）

| 来源 | 方式 | 效率 |
|------|------|------|
| 探索地图 | 随机掉落 | 中 |
| 回收武器 | 近战伤害/10，远程伤害/15 | 高（收集低级武器回收） |
| 流浪商人 | 交易 | 中 |
| 山洞探索 | 概率获取 | 低 |
| 柳如烟每日收获 | 随机 | 低 |

**总需求估算**：
- 周目一：25（城堡身份）+ 10（马三任务3）+ 50（10汽油×5香烟）= 85根
- 周目二：25（城堡身份）+ 10（国王任务3）= 35根
- 💡 优先回收武器获取香烟

### 抗感染血清

| 来源 | 方式 |
|------|------|
| 城郊大型医院 | 探索掉落 |
| 莉莉丝交易 | 医疗交易 |
| 皇后杂货铺 | 3皇家币/支 |
| 苏小涵任务 | 罐头收集任务奖励×3 |

**总需求估算**：
- 周目一：3（救柳如烟）+ 30（博士交易，如果做的话）= 33支
- 周目二：30（博士交易）= 30支

### 作物种子（沐苗苗任务3）

- 从沐苗苗处购买种子
- 在实验田种植，1-3天成熟
- 需要15种不同作物各1份
- 💡 尽早开始种植，每种作物种1份即可

### 汽油

- 来源1：城郊废弃加油站王铁柱，5香烟换1汽油
- 来源2：作弊码 `/get_gasoline_汽油_5`（每次获得5个）
- 周目一需要10个（柳如烟任务4）= 50香烟
- 周目二需要1个（游艇，可选）= 5香烟

### AWM（火箭链关键）

- 唯一来源：绝密航天基地太空箱，5%概率
- 每存档仅触发1次
- 💡 反复探索航天基地，这是最大的随机瓶颈
- 💡 提高遭遇率：确保装备足够好的武器和防具

## 常见问题

**Q：为什么不能1周目全收集？**
A：ending_prince（储君）需要杀首领，而ending_captain（队长）需要首领存活，两者互斥。同理，ending_newforce需要杀马三，ending_food需要马三存活。

**Q：ending_hope为什么放在周目二？**
A：ending_hope需要火箭链（城堡线）+ V/小寒/莉莉丝好感150。虽然持有城堡身份时无法与她们交互，但只要在获取城堡身份前刷满好感，好感度会保留，不影响ending_hope触发。

**Q：ending_999为什么放在周目二？**
A：火箭链的地下区域需要次元收纳背包（68格），而次元背包来自ending_999。所以必须先触发ending_999获取背包，才能完成火箭链。

**Q：AWM太难获取怎么办？**
A：AWM来自航天基地太空箱（5%概率），是最大的随机瓶颈。建议：装备最强武器+防具后反复探索航天基地，每次探索都有5%概率。耐心是关键。

**Q：存档切换会不会丢失进度？**
A：不会。结局一旦触发就被记录到unlockedEndings，即使读档也不会丢失。但注意不要覆盖关键存档位。

**Q：周目一需要做柳如烟任务链吗？**
A：需要救出柳如烟（ending_newforce条件之一），但不需要完成她的4段任务链（那是ending_elopement）。在存档切换阶段再完成柳如烟任务链。

**Q：周目二需要重新击杀马三吗？**
A：是的！每个周目是全新的游戏状态，所有条件需要重新满足。ending_newforce的6个条件都需要在本周目内完成。

**Q：崩溃值100%怎么办？**
A：崩溃满时无法睡觉、进食、饮水，也无法正常战斗。自救途径：使用止痛片（崩溃-5）、肾上腺素（崩溃-10）、手术包（崩溃-10）、战地医疗箱（崩溃-30）或万能针剂（崩溃-50）降低崩溃值。崩溃降至100以下后恢复正常。💡 游戏内会在拒绝进食/饮水时提示可用医疗物品自救。

**Q：黑市商人买弹药怎么选择数量？**
A：与黑市商人购买弹药时，可以选择用1根、5根、10根或全部香烟兑换，1根香烟≈3~6发弹药（随机）。不再是一次性消耗所有香烟。
