## 一、基础机制速览

### 时间系统

一天 = **8 回合**（凌晨→早上→上午→中午→下午→傍晚→夜晚→深夜）

### 核心属性

| 属性 | 说明 | 归零/上限惩罚 |
|:-----|:-----|:-----|
| ❤️ 生命值 (HP) | 上限 240 | 降至 0 = 死亡（结局一） |
| 🍖 饱食度 | 每回合 -4 | 降至 0 后每回合 -10 HP |
| 💧 水分 | 每回合 -4 | 降至 0 后每回合 -10 HP |
| 🦠 感染值 | 被丧尸命中 +2 | 达到 100 = 死亡（结局一） |
| 💔 崩溃值 | 每 16 回合清醒自动 +20 | ≥50 显示"崩溃"；≥100 无法战斗/进食/饮水/睡觉 |

### 战斗参数

| 参数 | 数值 |
|:-----|:-----|
| 逃跑成功率 | 25% |
| 远程闪避率 | 60%（远程战斗模式闪避丧尸攻击） |
| 暴击倍率 | 2× |
| 柳如烟协同攻击 | 30% 概率，造成 20 额外伤害 |
| 露露薇战后治疗 | 战斗胜利后自动 +20 HP |
| 感染值（被命中） | +2 / 次 |

### 睡眠

- 消耗 **4 回合**
- 恢复 HP **+20\~40**
- 降低崩溃 **-10\~25**
- 基地等级额外加成：+0 / +5 / +10 / +15 / +20 HP

### 新手必知

- **无怪图"曙光阵地"可无限生存**——睡觉、吃饭、喝水、打工全在这里
- **谷仓每 3 天可采摘水果**，农家乐有废弃菜田
- **瞭望塔登塔可减崩溃**
- **江边港口看景色也可减崩溃**
- **酸雨天无法出门**，提前规划

## 二、快速上手：苟活打工仔流程

> 适合：不想冒险、想安稳发育的玩家
> 难度：★☆☆☆☆（基本不会死）

### 核心循环（一天8回合）

```
打工(1回合) → 打工(2回合) → 打工(3回合) → 打工(4回合)
→ 乞讨/对话(5回合) → 吃饭喝水(6回合) → 睡觉(7-8回合)
```

### 消耗与收益计算

| 项目 | 消耗 | 补充方式 |
|:-----|:-----|:-----|
| 崩溃 +40（打工） | 睡觉恢复 | 睡觉一次 -10\~25 崩溃 |
| 水分 -18（打工） | 喝水 | 乞讨/阵地搜刮/仓库 |
| 饱食 -18（打工） | 吃饭 | 谷仓 3 天采摘水果、阵地食物 |

**结论：** 打工赚的物资远高于消耗，只要不出去打架，**永远不会死**。

### 打工路线图

1. 每天起床先 **乞讨**（阵地 NPC 对话有概率给水果）
2. **打工**（注意：黑夜不能打工）
3. 用赚来的物资 **吃饭喝水**
4. **睡觉** 跳过夜晚，恢复崩溃和血量
5. 每 3 天去一次 **谷仓采摘水果**
6. 攒够一定实力再去探索

## 三、完整攻略·新兵到老兵

### 第一阶段：开局求生（1\~5天）

**目标：** 活下来 + 拿到基础武器

#### 第1天

1. ✅ **去曙光阵地找 V 小姐对话**，领取初见任务（1 食物 + 1 饮料）
2. ✅ 完成任务获得 **小匕首**——你的第一把像样的武器
3. ✅ 去**乡村废弃谷仓**（遇敌率仅 5%）搜刮食物和武器
4. ✅ 注意血量，不舒服就回家睡觉

#### 第2\~5天

- 🔄 每天循环：打工 → 吃饭 → 睡觉
- 🏪 每 3 天去谷仓**采摘水果**（稳定食物来源）
- 🎒 去**河边露营地**（遇敌率 12%）搜刮背包，目标是 **28 格登山包**以上
- ⚔️ 谷仓有概率掉落近战武器，搜到**铁铲 / 撬棍**就换掉小匕首

#### 武器过渡路线

```
拳头(10) → 小匕首(40) → 铁铲(48)/撬棍(60)/消防斧(68)
→ 指虎刀(66)/大砍刀(75) → 西洋剑(79)/龙抄剑(88)/长骑枪(90)
→ 武士刀(99)/唐横刀(102)/电锯(105) → 寰宇之剑(118)
```

***

### 第二阶段：积累期（5\~15天）

**目标：** 攒物资、升装备、准备救伙伴

#### 推荐刷图顺序

| 地图 | 遇敌率 | 主要产出 |
|:-----|:-----:|:-----|
| 乡村废弃谷仓 | 5% | 食物 + 武器 |
| 深山农家乐村落 | 10% | 食物 + 水果 + 种子 |
| 河边露营地 | 12% | 背包 + 药品 |
| 高校大学城 | 15% | 食物 + 背包 |
| 城郊废弃加油站 | 18% | 弹药 + 武器 |

#### 关键行动

1. 🎒 **找大背包**：露营地搜刮尸体、瞭望塔（出 64 格超重大型负重背包）
2. ⚔️ **刷武器**：加油站、工业园区（30% 近战率）——**工业园区是刷近战武器的圣地！**
3. 💊 **囤药品**：重点囤**抗感染血清**和**止血带**（后面任务大量需要）
4. 🚬 **攒香烟**：香烟是硬通货！莉莉丝回收近战武器给烟，黑市商人用烟换一切

#### 打不过怎么办？

- 血量不健康就**回家睡觉**，别硬撑
- 崩溃高了去**瞭望塔登塔**（减崩溃）或**江边港口看景色**
- 带足食物和水再出门

### 第三阶段：伙伴招募（15\~30天）

**目标：** 30 天内救下柳如烟 + 招募露露薇

>- ⚠️ **柳如烟有时限：30 天后她会彻底变成丧尸，无法再救！**
- 💡 发现柳如烟后，生存笔记中会自动添加救援倒计时提醒

#### 前置：获得 60+ 伤害武器

- 去**谷仓 / 加油站 / 工业园区**刷武器
- 或者找 V 小姐做任务拿好武器

#### 救柳如烟（限时！）

```
市中心综合商场（遇敌率 35%）
  → 找到半感染女人（柳如烟）
  → 需要 3 支抗感染血清 才能救她
  → 成功救下 → 触发长剧情 → 带回家 → 获得第一个伙伴
```

**血清获取途径：**

| 方式 | 说明 |
|:-----|:-----|
| 🏥 城郊大型医院（遇敌率 50%） | 40% 概率掉药品，刷得越快越好 |
| 🛒 黑市商人 | 随机刷新，偶尔有血清 |
| 💊 搜刮中高难度图 | 警察局、研究所等会出 |

**刷医院技巧：**

- 别贪，打一轮就回家睡觉，睡醒再来
- 带足食物和水
- 有 60+ 伤害武器再去，不然打不过军警丧尸（160 HP / 36 伤）

#### 柳如烟任务链（救回后解锁）

救回柳如烟后，前往**高校大学城→一间旧教室**可开启柳如烟专属任务链，共 4 段任务，最终可触发**结局十二"私奔"**。详见[柳如烟大学任务链](#柳如烟大学任务链)章节。

#### 招募露露薇（无时限）

```
城郊大型医院（遇敌率 50%）
  → 找到理智的护士丧尸（露露薇）
  → 投喂罐头食品（黄桃/豆类/牛肉/军粮/沙丁鱼/金枪鱼/秋刀鱼/青花鱼罐头）
  → 好感度达到 150 → 带回家 → 获得第二个伙伴
```

**技巧：** 罐头在谷仓、超市、农家乐都能刷到。每次投喂 +2\~3 好感，大概需要 50\~75 个罐头。

#### 伙伴作用

| 伙伴 | 获取条件 | 每日产出 / 战斗效果 |
|:----:|:-----|:-----|
| 🌸 柳如烟 | 3 血清（限 30 天） | 每日 2 随机物资（食物 / 饮料 / 水果 / 弹药） |
| 🏥 露露薇 | 150 好感（不限时） | 每日 1 药品（创可贴\~肾上腺素） |
| ⚔️ 柳如烟（战斗） | 已招募 | 战斗中 30% 概率协同攻击，造成 20 额外伤害 |
| 💉 露露薇（战斗后） | 已招募 | 战斗胜利后自动回血 +20 HP |

#### BOSS 打法建议

**马三（高速服务区）：**
- 有 70+ 近战武器或手枪再去
- 带好药，他的远程先手可能直接打残你
- 打赢后 AK47 直接让你进入热武器时代

**黑影（仓储超市）：**
- 比马三还难！建议有龙抄剑或枪械再去
- 闪避率 50%，远程攻击命中率很低
- 打赢后 GP100 是强力左轮

**丧尸之王（丧尸巢穴）：**
- **游戏最强丧尸敌人，没有之一**
- 115 伤害，两下就能送走满血玩家
- 建议带满战地医疗箱 + 满耐久近战武器 + 远程武器
- 推荐装备：M4A1 + 5.56 弹药充足 + 寰宇之剑 / 电锯
- 打赢后获得寂灭徽章，是触发"成为新势力"结局的关键条件之一

**银行行长（末日城堡）：**
- 借贷逾期不还才会触发战斗
- 350 HP / 30% 闪避，难度中等
- 击杀后债务清除 + 获得武士刀（99 伤 / epic）

#### 研究所终极交易：陈博士

```
生化研究所（遇敌率 70%）
  → 上交 30 支抗感染血清
  → 获得 M700 狙击枪(120 伤) + 7.62×51mm ×30
```

**M700 是游戏最强可获取远程武器之一**，仅次于 AWM（200 伤）。

**陈博士火箭任务链前置条件：** 需先完成博士血清交易 + 击败航天基地的巨型丧尸傀儡。💡 如果前置条件未满足，游戏内会在博士的对话选项中显示具体缺失条件。

详见下方[结局系统](#结局系统)章节。完成所有主线任务后可触发"成为新势力"结局，获得终极近战武器 **无畏之刃(130 伤 / 20% 连击 / legendary)**。

### 💰 香烟经济

**香烟就是末日货币！** 所有高级交易都需要香烟。

#### 核心循环：香烟 → 物资 → 香烟

```
搜刮武器 → 莉莉丝回收 → 获得香烟
    ↓
香烟 → 黑市商人 → 买弹药 / 武器 / 种子
    ↓
更好的武器 → 挑战高危图 → 更多武器 / 物资
    ↓
多余武器 → 莉莉丝回收 → 更多香烟
    ↑_________________________|
```

#### 香烟获取途径汇总

| 途径 | 收益 | 稳定性 |
|:-----|:-----|:-----:|
| 莉莉丝回收近战武器 | 1 烟 / 把 | ⭐⭐⭐ 后期武器溢出，稳定变现 |
| 莉莉丝回收远程武器 | 2 烟 / 把 | ⭐⭐⭐ 远程更值钱 |
| 搜刮工业园 / 工厂仓库 | 10% 出烟 | ⭐⭐ 概率较低 |
| 击杀黑市商人 | 2\~5 烟 | ⭐ 有风险 |
| 城堡打工 | 1 烟 / 次 | ⭐ 消耗大(18 饱食 + 18 水分 + 40 崩溃) |
| 城堡觐见（爵位 5+） | 2 烟 / 天 | ⭐⭐⭐ 稳定被动收入 |
| 莉莉丝 150 好感奖励 | 20 烟（一次性） | ⭐⭐⭐ 一次性大额 |

#### 香烟消费优先级

1. **清除感染(10 烟)** — 感染值接近 100 时必须清，否则死亡
2. **维修枪支(10 烟)** — 主力武器坏了很危险
3. **黑市买弹药(1 烟 / 3\~6 发)** — 维持远程火力
4. **黑市买武器(10\~15 烟)** — 升级装备
5. **修复弓弩(5 烟)** — 比枪便宜，日常清怪用
6. **买种子(1 烟 / 包)** — 基地种植长期收益

**省钱技巧：** 打小怪用弓弩（零维修费），打 BOSS / 强敌才用枪械，最大化香烟利用率。

## 十、常见问题

**Q：开局怎么玩？**
A：去谷仓（安全）搜刮 → 找 V 小姐拿小匕首 → 打工攒钱 → 逐步探索低危图。

**Q：柳如烟没救到怎么办？**
A：30 天以后她会彻底变成丧尸，只能读档或者重新开局。所以**一定要在 30 天内凑够 3 支血清**。

**Q：背包满了怎么办？**
A：造仓库（基地建设 → 建造仓库），把不用的物资存进去。仓库最高 100 格。

**Q：怎么快速刷好感度？**
A：每天对话 4 次（每次 +1 好感），送礼（食物 / 饮料 +3，药品 +5，香烟 +2）。莉莉丝送礼加成更高（食物 +6，药品 +10）。

**Q：感染值高了怎么办？**
A：用清创药(-25) / 抗生素(-25) / 抗感染血清(-60) / 改良抗体针(-90)，或者找莉莉丝花 10 烟清除 -50 感染。城堡爵位高也有感染治疗比例。

**Q：崩溃值高了怎么办？**
A：睡觉(-10\~25)、止痛片(-5) / 肾上腺素(-10) / 手术包(-10) / 战地医疗箱(-30) / 万能针剂(-50)、瞭望塔登塔(-10)、江边看景色(-10)、城堡舞会(-50)、百味啤酒(-10) / 动力啤酒(-10) / 高度白酒(-30)。

**Q：近战武器坏了怎么办？**
A：莉莉丝只修**远程武器**。近战武器坏了就换一把新的。好在近战武器掉落很多。

**Q：远程武器完整度低了怎么办？**
A：找莉莉丝花 10 烟维修(枪械)或 5 烟维修(弓弩)。弓弩**不消耗完整度**，打小怪用弓省维修费。

**Q：最强的武器是什么？**
A：近战是尚方宝剑(250 伤 / 1 耐久 / legendary，一击即碎)和无畏之刃(130 伤 / 100 耐久 / 20% 连击 / legendary，实用最强)；远程是 AWM(200 伤 / legendary)。无畏之刃通过"成为新势力"结局获得。此外还有剧情专属武器：原野之弓(52 伤 / 52% 暴击 / legendary)、皇家短枪(99 伤 / legendary)、皇家银剑(88 伤 / 188 耐久 / 20% 连击 / legendary)。

**Q：游戏有几种结局？**
A：游戏共有 **12 种结局**：死亡(Game Over)、未知的将来(999 天)、权力的沉溺(储君身份牌)、发光发热吧(曙光队长徽章)、成为新势力(6 条件)、飞向太空(独自火箭 / Game Over)、希望的火种(团队火箭 / Game Over)、留下来吧(博士离开)、我不是农神(沐苗苗任务三)、爱恨情仇(皇后任务四)、沦为盘中餐(马三任务四接受 / Game Over)、私奔(柳如烟任务四 / Game Over)。其中结局 2\~5 和 8\~10 可继续游戏。

**Q：AWM 怎么获得？**
A：AWM(200 伤 / legendary)是游戏最强远程武器，只能通过**绝密航天基地**5% 概率发现坠毁太空舱获得（仅 1 次机会）。绝密航天基地遇敌率高达 90%，建议带满药品和顶级装备再去。正常游戏极难获得，属于终极收藏品。

**Q：怎么加入曙光阵地？**
A：在曙光阵地找到**阵地首领**，选择"加入阵地"即可获得**曙光徽章**。注意：持有贵族身份牌时无法加入（首领会拒绝），需要先注销贵族身份。

**Q：贵族身份有什么用？**
A：贵族身份牌(25 烟)可以进入**末日城堡内部**，享受以下特权：参加贵族宴会(饱食+50 / 水分+50)、参加贵族舞会(崩溃-50)、在贵族客房休息(HP+50)。提升爵位后还有感染治疗(最高 80%)、每日觐见 2 烟、花园随机药品等。但持有贵族身份会失去曙光阵地 NPC 互动权限，且无法加入曙光阵地。

**Q：城堡债务逾期会怎样？**
A：逾期未还(超过 10 天)会触发**没收全部物品**——身上所有东西被扒光！这个惩罚只触发一次，还清债务后恢复正常。建议借贷前确保 10 天内能还清，或在逾期前找银行行长宽限 5 天。

**Q：双身份冲突怎么处理？**
A：游戏**不允许同时持有曙光徽章和贵族身份牌**。如果你已有曙光徽章再去办贵族身份，系统会自动清除贵族身份牌、保留曙光徽章。如果已有贵族身份牌去加入曙光阵地，阵地首领会拒绝互动。建议先确定走哪条路线再选择身份。

**Q：皇家币怎么获得？**
A：完成皇后任务线（需先完成阵地首领任务链获取定情信物）可获得皇家币奖励。任务四完成后解锁"皇家交易"，可用枸杞(2→1 币)和西洋剑(1→5 币)兑换皇家币。

**Q：城堡通行证有什么用？**
A：城堡通行证（皇后任务四奖励）让你可以直接通过守卫进入城堡，无需身份牌或贿赂。它**不会**触发曙光阵地的排斥，也不会因为持有曙光队长徽章而消失。

**Q：苗苗日记本怎么获得？**
A：前往山顶废弃瞭望塔，选择"花圈墓碑"→"查看"（需农业大师徽章）→"挖掘"（需装备铁铲），即可获得苗苗日记本。持有日记本后与沐苗苗对话可解锁隐藏任务线。

**Q：感染值每次被丧尸打加多少？**
A：每次被丧尸命中 +2 感染值。感染值达到 100 即死亡，务必及时用清创药 / 血清 / 抗体针治疗。

**Q：崩溃值 100 会怎样？**
A：崩溃值达到 50 会显示"崩溃"状态，达到 **100 则无法战斗、进食、饮水、睡觉**。每 16 回合清醒自动 +20 崩溃，注意定期睡觉或使用减崩溃物品。💡 崩溃满时，游戏会提示可使用止痛片(崩溃-5)、肾上腺素(崩溃-10)、手术包(崩溃-10)、战地医疗箱(崩溃-30)或万能针剂(崩溃-50)恢复精神状态。崩溃降至100以下后恢复正常。

**Q：弓弩真的不消耗完整度吗？**
A：是的！弓和弩**不消耗完整度**，但它们有暴击率（非 0%），适合日常清怪省维修费。枪械每次射击 -1 完整度，需定期找莉莉丝维修。

**Q：马三的任务怎么解锁？**
A：先去国道高速服务区的餐厅吃一次"杂肉"（⚠️ 每次食用+10感染值），然后和马三对话选择"关于餐厅的食物"。注意：马三任务4无论接受还是击败马三，都会触发结局十一"沦为盘中餐"（终局），这是不可逆的选择。

**Q：汽油怎么获得？**
A：去城郊废弃加油站找王铁柱，5 根香烟换 1 个汽油。也可以使用作弊码 `/get_gasoline_汽油_5`（每次获得5个）。

**Q：柳如烟的任务线怎么开启？**
A：先救出柳如烟（商场 3 血清），然后去高校大学城的"一间旧教室"。

**Q：怎么去小型岛屿？**
A：需要城堡身份牌 + 1 个汽油（汽油可通过王铁柱用香烟兑换），从江边港口码头乘坐城堡游艇前往。

**Q：岛屿银行怎么借贷？**
A：需要城堡通行证或城堡身份牌，可借 10 / 20 / 30 皇家币，30 天还款期限，逾期追加 10% 利息。

**Q：崩溃值满了遇到丧尸怎么办？**
A：崩溃值100%时进入战斗无法攻击，只能被动挨打。游戏会在战前显示⚠警告。建议先使用止痛片/肾上腺素等降低崩溃值的药品恢复精神，再进行战斗。

**Q：黑市商人买弹药可以选数量吗？**
A：可以！购买弹药时可以选择用1根、5根、10根或全部香烟兑换，1根香烟≈3~6发弹药（随机）。不再是一次性消耗所有香烟。

**Q：新生力量结局怎么追踪进度？**
A：完成任意一个前置条件后，生存笔记中会自动添加"新生力量"进度条目，显示6个条件的完成情况（X/6）。

**Q：火箭"带上重要的人"需要什么条件？**
A：需要V小姐、苏小涵、莉莉丝三人的好感度都达到150。选择该选项时，游戏会显示当前好感度进度（如 V:80/150 小涵:60/150 莉莉:150/150），方便判断还差多少。
